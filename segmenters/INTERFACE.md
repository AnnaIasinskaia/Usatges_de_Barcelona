# Контракт сегментеров

## Назначение

Документ описывает унифицированный интерфейс (контракт) для всех модулей сегментации, расположенных в папке `segmenters/`. Цель контракта — обеспечить совместимость сегментеров с основным пайплайном обработки исторических текстов (`pipeline.py`, `pipeline_gramoty.py`) и упростить интеграцию новых источников.

Контракт определяет:
- сигнатуру функции сегментации;
- ожидаемые параметры;
- формат возвращаемых данных;
- требования к валидации и обработке ошибок.
